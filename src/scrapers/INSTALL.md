# Install and run
```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

this will generate the `helpradar.db` sqlite file. No sqlite3? Don't worry, you can upload the database to https://sqliteonline.com/ and view it there!
